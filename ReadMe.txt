This is a POC project and the purpose of this is to demonstrate the usage of activeMQ. 
Later, the same scenario will be simplified using Appache Camel content based routing.  


This contains the set of instructions required to build and run ActiveMq application.


Building the Project.
==================== 

Pre-requisites
--------------
JDK 1.6.x or higher
Maven 2 or higher
java_home and m2_home has been set.
ActiveMQ message broker is configured and running on localhost:61616 

Steps:
------
1 - download and unzip the project.
2 - Go to the project home directory of activeMQ, and run 
    $ mvn clean package

Running the application.
========================

go to ContextLoader.java and run as an application.
